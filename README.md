# 🛡️ AI Phishing Email Analyzer

An AI-powered phishing detection dashboard built with Python, Streamlit, and Ollama.

---

## Features

- Rule-based phishing detection
- AI-generated email analysis
- URL extraction
- Threat classification
- File upload support
- Local LLM integration
- Cyber-themed SOC dashboard
